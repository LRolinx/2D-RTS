# 2D RTS

一个基于 Vue 3 + TypeScript + Vite + PixiJS 的 2D RTS 原型。当前版本已经从旧的 p5 直挂脚本迁移到 PixiJS 渲染内核，Vue 负责应用生命周期，PixiJS 负责战场渲染、单位、路径、战斗和交互。

## 运行

```bash
pnpm install
pnpm dev
```

默认本地地址：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
pnpm build
```

## 当前玩法

- 左键点选单位或地块
- 左键拖拽框选多个单位
- 右键给选中单位下达编队移动命令
- 玩家和敌方都有基地与混合兵种部队
- 侦察车、突击坦克、等离子坦克、猛犸重坦、武装直升机具备不同生命、速度、射程、伤害和冷却
- 单位会自动索敌、转向、开火、发射投射物并造成伤害
- 地图上有可争夺资源点，单位驻守附近会持续采集
- 基地会周期性消耗资源生产增援
- Q / W / E / A / S 可以手动生产侦察车、突击坦克、等离子坦克、猛犸重坦、武装直升机
- 选中基地附近地块后按 T 可以建造防御塔
- 防御塔会自动索敌并参与基地防御
- 单位死亡会留下残骸并触发爆炸反馈
- 鼠标滚轮缩放战场
- 鼠标中键或 Alt + 拖拽移动视角
- Space 暂停/继续
- R 重置战场
- 1 / 2 / 3 切换模拟速度
